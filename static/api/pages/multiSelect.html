说明

多选

使用时请用 me.require 方法按需加载到页面，（此组件需引入css）,并且额外要引入meScroll组件

请注意：

HTML
<select name="" class="multiple">
	<option value="sex1" title = "1">option1</option>
	<option value="sex2" title = "2">option2option2option2option2option2option2option2</option>
	<option value="sex3" title = "3">option3</option>
	<option value="sex4" title = "4">option4</option>
	<option value="sex5" title = "5">option5</option>
	<option value="sex6" title = "6">option6</option>
	<option value="sex4" title = "4">option7</option>
	<option value="sex5" title = "5">option8</option>
	<option value="sex6" title = "6">option9</option>
</select>
JS
me.multiSelect({
      ele:'#multiple',
      placeholder:'please',
      getValue:function(data){
          console.log(data)
      },
      item_choice_color:'pink'
      
  })

示例

API

多选框（注：若页面中多次调用本方法，需按照dom的书写select顺序来调用此方法，否则会影响渲染效果;）

名称 说明 使用示例
ele（必传参数） 需替换的select的id或者class 
me.multiSelect({
  ele:'#multiple'
})
placeholder 提示信息 
me.multiSelect({
  ele:'#multiple',
  placeholder:'请选择'
})
getValue 点击叉号、复选框、全选、反选，在input框中按退格键的回调函数，该方法返回值为数组类型，数组项为json格式，所选的值 
me.multiSelect({
  ele:'#multiple',
  placeholder:'请选择',
  getValue:function(data){
  console.log(data)
 }
})
item_choice_color 复选框及单选按钮的颜色
(根据me组件库的checkbox和radio的颜色传参数，eg:'blue')		 
me.multiSelect({
  ele:'#multiple',
  placeholder:'请选择',
  getValue:function(data){
  console.log(data)
 }
 item_choice_color:blue
})
hover 鼠标经过下拉列表项时的样式设置(所传类型为对象格式) 
me.multiSelect({
  ele:'#multiple',
  placeholder:'请选择',
 hover:{'background-color':'#a08ee0'},
 getValue:function(data){
  console.log(data)
 },
 item_choice_color:blue
 })
list_item_style 下拉列表项的样式设置(所传类型为对象格式) 
me.multiSelect({
  ele:'#multiple',
  placeholder:'请选择',
 hover:{'background-color':'#a08ee0'},
 getValue:function(data){
  console.log(data)
 },
 item_choice_color:blue,
 list_item_style:{'font-size':'15px'})
menu 下拉列表框的样式设置(所传类型为对象格式) 
me.multiSelect({
  ele:'#multiple',
  placeholder:'请选择',
 hover:{'background-color':'#a08ee0'},
 getValue:function(data){
  console.log(data)
 },
 item_choice_color:blue,
 list_item_style:{'font-size':'15px'},
 menu:{'width':'200px'}})
input_style 输入框的样式设置(所传类型为对象格式) 
me.multiSelect({
  ele:'#multiple',
  placeholder:'请选择',
 hover:{'background-color':'#a08ee0'},
 getValue:function(data){
  console.log(data)
 },
 item_choice_color:blue,
 list_item_style:{'font-size':'15px'},
 menu:{'width':'200px'},
 input_style:{'border':'1px solid red'}
})
