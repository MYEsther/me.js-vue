说明

progress控件主要用于展示项目进度

使用时请用 me.require 方法按需加载到页面

HTML

进度条约定相关html结构，样式可以根据实际情况用css样式定义。

<div class="me_progress" percent=100 radius="border-radius" move animation> <!--被渲染菜单class-->
</div>

JS

需要通过me.require导入menu组件js、css文件。具体见me.require用法

me.require(['progress&css'], 'progress'); //加载progress.js、progress.css文件
me.require.ready('progress',function(){
  me.menu.render({
          backgroundColor:"red",        //进度颜色
          strokeWidth:"6",              //进度条高度
          isShowText:true,              //是否显示文字
          location:"right",             //文字显示位置
          width:100,                    //进度宽度
          sucBackgroundColor:"green"    //100%状态下显示的进度颜色
          },
          function(){},                 //回调函数，进度条加载完成后
          2);                           //用到模板加载多条数据时，需要传几条数据
});

示例 type='line' 属性添加radius="border-radius"为圆角 percent=30百分比 进度条不运动

示例 type='line' 进度条运动 属性添加move加上运动 animation加上进度效果

示例 type='circle' 进度条运动

示例 type='circle' 进度条运动并100变色

参数

名称 类型 可选 默认值 描述
o 对象 否 
parentBackgroundColor = o.parentBackgroundColor || "#eee", //背景色
backgroundColor = o.backgroundColor || "red",     //前景色
strokeWidth = o.strokeWidth || "6",      //高度
width = o.width || 200,        //进度条宽度
isShowText = o.isShowText ||　false, //是否显示文字描述
text = o.text || percent+"%",       //文字描述 
location = o.location ||　"inside", //文字描述的位置
successText = o.successText || text;   //100%状态下的文字描述
sucBackgroundColor = o.sucBackgroundColor || backgroundColor; //100%状态下的前景色
进度条的颜色，宽度，高度，是否显示文字，文字位置等
callback 方法 是 进度条加载完后执行
l 变量（数字） 是

如果是模板渲染加载时需要传条数
